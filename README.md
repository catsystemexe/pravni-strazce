# e-advokat-core (skeleton)

Skeleton architektury pro právního AI agenta (core část, bez frontendu a DB).
Tento projekt je připravený jako základ pro postupné doplňování logiky, dle návrhu:
- runtime orchestrátor
- modulární engines
- domain packs
- šablony výstupů
- produktová konfigurace (e-advokat PRO)
